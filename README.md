似乎 _通信、空天、卫星、数据_ ，这些词才是这些方向中的高频词

这也正是我想表达的，虽然院校建立是对半分，但总体上来说会更偏**电子信息**

这也就意味着，如果一个单纯想做传统安全（系统安全、软件安全等等）的学生进入网安专业，大概率会上一些可能毫无作用的课；而对于想做IoT或者通信安全、密码学的学生会更适应这个专业的氛围
