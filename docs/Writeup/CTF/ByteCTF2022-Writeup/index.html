同态性验证​

Paillier是加法同态

D((c1⋅c2)mod  n2)=D(gm1rn⋅gm2rnmod  n2)=D(gm1+m2(r2)nmod  n2)=m1+m2\begin{aligned} D((c_1 \cdot c_2) \mod{n^2}) & = D(g^{m_1} r^{n} \cdot g^{m_2} r^{n} \mod{n^2})\\ & = D(g^{m_1 + m_2} (r^2)^n \mod{n^2})\\ & = m_1 + m_2 \end{aligned}D((c1​⋅c2​)modn2)​=D(gm1​rn⋅gm2​rnmodn2)=D(gm1​+m2​(r2)nmodn2)=m1​+m2​​

解题过程​

程序的整体逻辑就是比较100轮大小，如果全胜就可以拿到flag

def round(expr):

    q = getPrime(N)

    n = p * q

    msg = int(input("msg = "))

    l = int(lcm(p - 1, q - 1))
    u = inverse(Fraction(pow(g, l, n * n) - 1, n), n)

    return (a > b) is bool(eval(expr, None, {'MSG': decode(msg, n, l, u)}))

输入一个表达式，每轮比较根据表达式的内容进行比较，如果表达式的真值和a > b相同即胜利

因为服务端设置的N = 512，所以让expr = "MSG < (2**512)"

根据Paillier的加法同态，让msg等于a-b，即 msg=D(a)D−1(b)mod  n2{\rm msg} = D(a)D^{-1}(b) \mod{n^2}msg=D(a)D−1(b)modn2 ，这样解密之后得到的就是 a−bmod  n2{a - b \mod{n^2}}a−bmodn2

  • 如果a > b，则a - b > 0，自然((a > b) is True) = True
  • 如果a < b，则a - b < 0，后面eval计算的表达式是在负数模了一个 n2n^2n2 之后位数到了1024位，自然((a > b) is False) = True

from Crypto.Util.number import getPrime, getRandomNBitInteger, inverse
from fractions import Fraction
from gmpy2 import lcm, invert

N = 512


def encode(m, n, g):
    r = getRandomNBitInteger(N)
    c = pow(g, m, n*n) * pow(r, n, n*n) % (n*n)
    return c


def decode(c, n, l, u):
    return int(Fraction(pow(c, l, n * n) - 1, n) * u % n)


def round(expr):
    p = getPrime(N)
    q = getPrime(N)

    n = p * q
    g = getRandomNBitInteger(N)
    print('n =', n)
    print('g =', g)

    a = getRandomNBitInteger(N)
    b = getRandomNBitInteger(N)

    print('a =', encode(a, n, g))
    print('b =', encode(b, n, g))

    new_a = encode(a, n, g)
    new_b = encode(b, n, g)

    # msg = new_a * invert(new_b, new_n**2) % (new_n**2)
    msg = new_a * invert(new_b, n**2) % (n**2)
    print('msg =', msg)

    l = int(lcm(p - 1, q - 1))
    u = inverse(Fraction(pow(g, l, n * n) - 1, n), n)

    plain = decode(msg, n, l, u)
    print("plaintext(a - b) = ", plain)

    return (a > b) is bool(eval(expr, None, {'MSG': decode(msg, n, l, u)}))

if __name__ == "__main__":
    expr = "MSG < (2**512)"
    print('Hello, Give me your expr:', expr)
    for i in range(100):
        print('Round:', i)
        try:
            assert round(expr)
        except:
            print('You lost.')
            break
